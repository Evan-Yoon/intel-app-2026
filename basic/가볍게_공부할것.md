# HTML/CSS/JS 가볍게 공부할 것

짧게 보고 바로 실습할 수 있는 순서로 정리했습니다.

## 1. HTML (기초 다지기)

- 시맨틱 태그: `header`, `main`, `section`, `article`, `footer`
- 폼 태그: `form`, `input`, `label`, `button`
- 테이블 태그: `table`, `tr`, `th`, `td`
- 접근성 기초: `alt`, `label for`, 제목 계층(`h1`~`h3`)

실습:
- `03_profile.html` 만들기
- 내 소개(이름/한 줄 소개/취미 목록) + 간단한 연락 폼 추가

## 2. CSS (눈에 보이는 변화)

- 선택자: 태그, 클래스(`.`), 아이디(`#`)
- 박스 모델: `margin`, `padding`, `border`
- 배치: `display`, `flex`, `justify-content`, `align-items`
- 텍스트/색상: `font-size`, `line-height`, `color`, `background`

실습:
- `style.css` 분리해서 적용하기(인라인 스타일 제거)
- 카드 UI 3개 가로 정렬(`flex`) 만들기

## 3. JavaScript (동작 추가)

- 변수: `let`, `const`
- 조건문/반복문: `if`, `for`
- DOM 선택/수정: `querySelector`, `textContent`, `classList`
- 이벤트: `click`, `input`

실습:
- 버튼 누르면 문구 바뀌기
- 입력창에 글자 입력하면 실시간 미리보기 만들기

## 4. 미니 프로젝트 (가볍게 완성)

주제: `오늘 할 일(To-do) 미니 앱`

기능:
- 할 일 추가
- 완료 체크(스타일 변경)
- 삭제 버튼

파일 구성 추천:
- `todo.html`
- `todo.css`
- `todo.js`

## 5. 추천 학습 루틴 (하루 30~40분)

- 10분: 개념 1개 읽기
- 20분: 직접 코드 치기
- 10분: 기존 코드 리팩터링(이름/구조/스타일 정리)
